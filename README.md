# avaliacao2Web


O MAVI teve o intuito de facilitar o processo de instituições, como controle de alunos, cursos, matriculas, e futuramente com novas funcionalidades.

No mavi temos a tela principal, onde sera a tela de login do administrador, logo após o login ser efetuado e ter ocorrido corretamente, ele ira redirecionar para a tela de apresentação, onde teremos o menu com as telas de Cursos, onde voce cadastra os cursos ou matérias que sua instituição tem ou terá, a tela de alunos onde o administrador do sistema ou o usuario de gestão cadastrais irá fazer o cadastro do aluno com suas informações, e a tela de matricula onde será vinculado um aluno a uma materia ja cadastrada.
